Παρουσίαση από τον κ. Πέτρο Αμοιρίδη, Μanager στην πλατφόρμα GitHub

Friday, December 2, 2016 - 16:45

Τη Δευτέρα 5/12/16, ώρα 9-11πμ στον ημιόροφο του Βιολογικού - Αίθουσα Β θα γίνει παρουσίαση από τον κ. Αμοιρίδη Πέτρο, Manager at GitHub με τίτλο:
"Contribute effectively to open source on GitHub using Git"

Περίληψη: "Contributing to open source is both a way to learn, but also a way to prove yourself to future employers. Of course, it's also a way to give back to the community, and improve the world. GitHub is the largest open source platform in the world. Learn how to contribute to open source projects using the GitHub Flow. This two hour presentation will show you how to use GitHub and Git to propose changes to open source projects using Pull Requests. It will also suggest a few ways to go about finding a good open source project to contribute to. GitHub Flow applies to closed source projects as well."

Στοιχεία επικοινωνίας: @amiridis στο Twitter - @petros στο GitHub 

Η διάλεξη γίνεται στα πλαίσια του μαθήματος Πληροφοριακά Συστήματα Παγκόσμιου Ιστού και η βοηθός του μαθήματος  Αντιγόνη Φούντα θα συντονίσει την παρουσίαση.
